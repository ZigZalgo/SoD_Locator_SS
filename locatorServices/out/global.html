dropData

"dropData" listener handles when a person performs release handgesture and drop the data associate with the person to the location this function will check whether there is a dataPoint within the dropRange if there is a dataPoint within the range the data associate with the person will be dump into the dataPoint otherWise a new dataPoint will spawn on the current location where the person is standing and link the data to it.
