Society of Devices

SoD (Society of Devices) is a framework which brings together multiple devices, allowing spatial awareness and communication amongst them.

Most part to this documentation is for all the socket.io requests. And the expected callback from those socket request. The documentation contains the sample code for socket emit on client side as well.
